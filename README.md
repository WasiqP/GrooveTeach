# AI Teacher Personal Assistant

**PulseBox** is an AI-powered personal assistant designed specifically for teachers to reduce administrative burden, streamline lesson planning, automate grading, and help teachers focus on what matters most - teaching.

## 🎯 The Problem

Australian teachers are the **2nd most stressed people in the world** because they spend too much time on:
- Student preparation & planning
- Administrative functions
- Marking attendance
- Grading assignments
- Report writing
- Parent communication

**These are tasks a Teacher's PA should handle - that's what PulseBox does!**

## ✨ Key Features

- 🤖 **AI Lesson Plan Generator** - Generate complete lesson plans in seconds
- 📊 **AI Grading Assistant** - Automate grading with AI feedback
- ✅ **Smart Attendance Tracker** - Quick attendance marking with analytics
- 📈 **Student Progress Dashboard** - Track individual student progress
- 📝 **AI Report Card Generator** - Auto-generate personalized report comments
- 📚 **Class Management** - Organize classes, students, and assignments
- 💬 **Parent Communication Hub** - Streamline parent-teacher communication
- 🎓 **Assignment Creator** - Create and manage assignments with AI assistance

## 🚀 Tech Stack

- **Frontend**: React Native (iOS & Android)
- **Backend**: Firebase (Firestore, Authentication, Cloud Functions)
- **AI**: OpenAI GPT-4 / Similar AI services
- **Navigation**: React Navigation
